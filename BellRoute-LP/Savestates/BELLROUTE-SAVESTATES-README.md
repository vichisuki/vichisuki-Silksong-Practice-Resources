# Bell Route LP Savestates

Description of each savestate created for Bell Route's Latest Patch in Silksong Any%.

ComSOB Sheet can be found [here](https://docs.google.com/spreadsheets/d/1-Hy5k_h9dBUUPlwh_I14K-DEwGcNdVQ0utI-KX56d8c/edit?gid=7316953#gid=7316953).

Click on the arrows next to each label for more info. 

### IL Pages
<details>
<summary><b>Page One</b></summary>

0. Moss Mother 
    - After bind prompt and movement is unlocked
    - If using with ComSOB Sheet, **please start split from game start**.
    - If using for practice, set start split to manual split and start timer at 21.90 for when movement starts
    - Start Split on Manual Split or Start New Game (Start)
    - End Split on Moss Mother (Transition)
1. Silk Spear
    - Underneath ladder prompt to exit Ruined Chapel
    - Press up to start split
    - Start Split on Moss Mother (Transition)
    - End Split on Silk Spear (Transition)
2. Bell Beast
    - Before silk wall to exit Silk Spear Room to include silk refund in IL practice
    - If you prefer not to do this, just hit the silk wall once before to get the refund
    - Start Split on Silk Spear (Transition)
    - End Split on Bell Beast (Transition)
3. Swift Step
    - Inside Bell Beast Silk Heart room just below Silk Heart pickup
    - Start Split on Quit to Menu/Bell Beast Transition
    - End Split on Swift Step (Transition)
4. Lace 1
    - Before Swift Step room exit
    - Start Split on Swift Step (Transition)
    - End Split on Lace 1 (Transition)
5. Drifter's Cloak
    - Outside Deep Docks bell room. Lever not yet hit
    - Start Split on Lace 1 Transition
    - End Split on Drifter's Cloak (Transition)
6. Greymoor Bell
    - Inside Seamstress room after obtaining Drifter's Cloak
    - Start Split on Drifter's Cloak (Transition)
    - End Split on Greymoor Bell (Event)
7. Moorwing
    - Inside Greymoor Bell room in front of lever
    - Start Split on Greymoor Bell (Event)
    - End Split on Moorwing (Transition)
8. Cling Grip
    - Moorwing room after boss dies
    - Start Split on Moorwing (Transition)
    - End Split on Cling Grip (Transition)
9. Widow
    - At the exit to Cling Grip Room
    - Start Split on Cling Grip (Transition)
    - End Split on Bellhart Bell (Event)
</details>

### Movement Pages
<details>
<summary><b>Page 4</b></summary>

0. Moss Mother
    - After bind prompt and movement is unlocked.
    - Functionally identical to IL - Moss Mother on Page 1.

1. Silk Spear
    - In Bone Bottom before transition to Marrow.

2. Bell Beast
    - After Silk Spear Transition in room with single Aknid
    - Note that this savestate gives you 5 silk, netting you no refund from the silk wall. Hit the aknid for an extra silk.

3. Swift Step
    - Room after Marrow Bell, right before transition to right side room (Bone_14).

4. Lace 1
    - Room after Swift Step with two elevators.

5. Drifter's Cloak
    - Deep Docks bell room after activating lever. Exit out via right transition.

6. Cloak Quest
    - Outside Seamstress room before room with Hokers in them. 
    - Seperate from Movement - Drifter's Cloak because Cloak Quest is important enough and RNG heavy to warrant its own savestate.

7. Greymoor Bell
    - Fourth Chorus room after skip.
    - To practice Fourth Chorus skip, use Trick - Fourth Chorus Skip on page 9

8. Moorwing
    - Room after Greymoor Bell activation.
    - This savestate encompasses both before and after the Greymoor arena. To practice afterwards, use the Boss - Greymoor Arena savestate and Kill All.

9. Cling Grip
    - Leftwards transition exiting Bellhart.
</details>

<details>
<summary><b>Page 5</b></summary>

0. Shellwood Bell
    - Leftwards transition exiting Bellhart.
    - This savestate has killed Widow and obtained Needolin, with Cling Grip.

1. Last Judge
    - First room after entering Blasted Steps.
    - All three tricks (heal skip, driznit pogo and walljump) are available on page 10.

2. Cogwork Dancers
    - Second room of Citadel with scale platforms.

3. Whispering Vaults Arena
    - Cogwork Dancers room after cutscene
    - Functionally identical to IL - Whispering Vaults Arena

4. Trobbio
    - Second room of Whispering Vaults.

5. Clawline
    - Bottom of elevator room with elevator already hit back up.

6. Post Clawline
    - Immediately after Clawline pickup, to practice movement upwards to Clawline arena.

7. Vaultkeeper's Melody
    - Room after Clawline arena, with Last Architect in it. Exit out right transition.

8. Architect's Melody
    - Room connecting Whispering Vaults and Cogwork Core. Exit out left transition.

9. Cogwork Core
    - Transition before upwards gauntlet to Architect's Melody.
    - This is still considered "Cogwork Core", and is labelled as such, as it is a subsplit of Architect's Melody, even though the player has accessed Cogwork Core previously.
    - Placed outside the room to preserve enemy cycles and movement.

</details>
